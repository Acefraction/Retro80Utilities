# パレットテンプレート仕様書

このフォルダには、Retro80Colorizerで使用される編集可能なパレットテンプレートが含まれています。

各パレットは **32×32 のグリッド構造を持つ PNG画像** として定義されます。  
**各セルは 16×16 ピクセル**で構成され、**各セルの中央のピクセル**（`x * 16 + 8`, `y * 16 + 8`）の色が、プログラムで実際に使用されるパレット色となります。

## 💡 構造

- **横方向（X軸）**は、同じ色グループ内の色バリエーション（例：オレンジ系の濃淡など）を表します。
- **縦方向（Y軸）**は、論理的な**機能グループ**を表します：
  - キャラクターの肌色
  - 背景用色
  - ハイライト / シャドウ
  - UI要素の色 など

最大で**32機能グループ × 各32色**まで定義可能です。

## 🖌️ 編集方法

- PNGファイルは、ドット単位で操作できる画像編集ツールで編集してください。
- 実際に使用されるのは**各セルの中央色のみ**です。まわりの色は人間向けの見やすさのためのものです。
- グリッド線はあっても構いません（プログラム側では無視されます）。

## 🔄 プログラム内での利用

- 実行時に、プログラムは各グリッドセルの中央ピクセル色だけを読み取って `Color[,]` 配列を構築します。
- これにより、視覚的に編集しやすく、かつ処理としてもシンプルに保たれます。
